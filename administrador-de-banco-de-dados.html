o que é?

Além da análise de dados, outra demanda cada vez maior devido ao aumento do uso da tecnologia e do fluxo de dados é por profissionais capazes de fazer uma boa administração e gerenciamento dos dados das organizações. São os administradores ou administradoras de banco de dados, também chamados de profissionais DBA, database administrator (do inglês, administradores de banco de dados).

Ocupando essa função, você deve manter uma base de dados robusta, otimizada e completa para consulta e análise de grandes quantidades de dados (Big Data), além de gerenciar, manter, instalar e monitorar os dados corporativos, sua integridade, eficiência e disponibilidade. Para atuar na área, normalmente é exigido que você tenha graduação em sistemas da informação ou ciência da computação, assim como conhecimentos de linguagens de banco de dados, como SQL, XQuery, OQL, GraphQL e LINQ.
